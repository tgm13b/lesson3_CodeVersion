Git Learning the language

Git is an open source version control system that can handle any type of project. A version control system allows people to handle, upload, and edit files from a remote repository. This allows a group of people to access and edit files that don't have to be on their local machine. It essentially takes screenshots of the work you do and keeps it all in place.

Remote Repository

Remote repositories are basically different versions of a project or file that is hosted somewhere on the Internet. These repositories are best suited for projects that a group of people will be working on. You are able to push and pull the data from these repositories and work on them.

GitHub Repository Hosting Service

GitHub is a repository hosting service where you can access a version control system and source code management. GitHub has almost 10 million users and you don't have to pay to become a member. Joining GitHub gives you access to remote repositories and is usually for the purpose of managing group projects. It is the largest host of source code throughout the entire world.
